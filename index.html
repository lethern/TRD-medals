How to find Bank?
navigate to folder
%USERPROFILE%\Documents\StarCraft II\Accounts\
then choose Account, then choose server (1-S2- is NA, 2-S2- is EU, 3-S2- is KR)
then go to \Banks\1-S2-1-2199346, (NA), \Banks\2-S2-1-6048667 (EU), \Banks\3-S2-1-4907429 (Asia)
the bank file is Bank1.SC2Bank
Every game, we start with 0 BattleXp, ServiceXp and TacticalXp.
We gain them usually by being active, the most probably by being the leader. Estimates:

ServiceXp: in general, using abilities
TacticalXp: again various actions, like searching cars or Bella' bones
BattleXp: 1 each time we shoot, 3 each time we kill something
